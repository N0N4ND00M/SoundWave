About the project

Built for Webbutveckling 1

SoundWave Pro är skapad som ett slutprojekt för Webbutveckling 1. Målet är att visa semantisk HTML, tydlig CSS-struktur, responsiv design och JavaScript som faktiskt manipulerar sidans innehåll och utseende.

Projektet använder en enda HTML-fil men simulerar flera sidor genom att visa och dölja sektioner med JavaScript. Det gör strukturen enkel men ändå interaktiv.

Implemented features

  • Responsiv layout för mobil, surfplatta och desktop
  • Hash-baserad navigation mellan simulerade sidor
  • JavaScript-renderade låtkort från datafil
  • Mood-filter med dynamisk visning
  • Fake music player som uppdateras vid klick
  • Favoritmarkering med räknare
  • Dark/light mode med localStorage
  • CSS-sprite för små ikoner

Device strategy

Mobile 375 × 667 px, en kolumn, hamburger-meny.
Tablet 1024 × 768 px, två kolumner och mer luftig layout.
Desktop Över 1024 px, sidebar/grid och mer avancerad ytfördelning.
